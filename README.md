### 项目执行命令

:fire: 
 
```bash
# 安装依赖
$ yarn install

# 启动项目（开发环境）
$ yarn dev

# 项目打包
$ yarn build
$ yarn start

# 项目纯静态打包（异步请求都没有）
$ yarn generate
```
